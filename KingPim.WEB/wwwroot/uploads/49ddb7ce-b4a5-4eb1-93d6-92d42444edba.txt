.Net Core och Raspbery Pi GPIO



Började för ett tag sedan bygga en robot med en Raspberry Pi vilket resulterade i att jag fick bygga mitt egna bibliotek fyllt med funktioner för att hantera pajens GPIO pinnar.

Nu känner jag att det vore riktigt skoj om flera skulle vilja haka på  och vidarutveckla detta bibliotek och kanske bidra med egen kod/funktioner eller varför inte slänga ihop egna projekt med detta som grund.

Jag har valt att döpa projektet till GpioCore och tanken är att detta ska vara helt open source så vem som helst kan bidra med egen kod eller använda detta samtidigt som jag vill försöka promota att allt fler provar på .Net Core
